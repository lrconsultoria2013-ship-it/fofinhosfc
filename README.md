# Fofinhos FC

Bem-vindo ao repositório oficial do **Fofinhos FC** - O Clube Mais Querido.

## Sobre o Projeto
Este é o website oficial do time, desenvolvido com:
- HTML5
- Tailwind CSS (via CDN)
- JavaScript

## Estrutura
- `index.html`: Página principal contendo toda a estrutura do site.
- `assets/`: Imagens e recursos (se houver).

## Como rodar localmente
Este é um site estático. Para rodar, basta abrir o arquivo `index.html` em seu navegador ou usar um servidor local simples como o Live Server do VS Code.

## Personalização
O arquivo `index.html` contém comentários detalhados sobre como alterar imagens, cores e textos.
